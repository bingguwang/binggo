
如何在windows下运行go程序，且不会打开窗口

一般在windows下运行的go exe程序都会打开一个窗口的
不想打开窗口可以做一下的尝试


##########################  方法 1: 使用 -w 和 -H 选项编译  ##########################
你可以在编译时使用 -w 和 -H 选项来生成一个控制台窗口不可见的可执行文件。运行以下命令：

go build -o myapp.exe -ldflags="-H=windowsgui" main.go
此命令将编译 main.go 文件，并生成一个名为 myapp.exe 的可执行文件，且不会打开控制台窗口


##########################  方法 2: 使用 cmd 的 start 命令  ##########################
如果你已经有一个控制台程序，但希望在执行时不弹出控制台窗口，可以通过在批处理文件中使用 start 命令来实现。

start /b myapp.exe

##########################  方法 3:  Go 应用程序封装为 Windows 服务  ##########################


当你希望将你的 Go 应用程序作为 Windows 服务来运行时，你可以使用 NSSM（Non-Sucking Service Manager）工具来实现。以下是详细的步骤：

步骤
下载 NSSM

首先，你需要下载 NSSM 工具。你可以从其官方网站 https://nssm.cc/download 下载最新版本的 nssm.exe。

安装应用程序

确保你的 Go 应用程序已经编译为可执行文件（.exe）。

打开命令提示符

在开始菜单中搜索“命令提示符”并打开。

使用 NSSM 安装服务

运行以下命令以使用 NSSM 安装你的 Go 应用程序作为服务。假设你的可执行文件为 myapp.exe，路径为 C:\path    o\myapp.exe。

nssm install MyGoService "C:\path    o\myapp.exe"
这将启动 NSSM 界面以配置服务。你可以在此界面上配置服务的名称、描述、启动参数等。

启动服务

安装完成后，你可以在 Windows 服务管理器中找到新创建的服务。选择你的服务，并点击“启动”按钮将其启动。

注意事项
在 NSSM 界面中，你可以配置服务的名称、描述、启动参数等。你也可以在 NSSM 的命令行模式下执行配置。

如果你的应用程序需要特定的环境变量或工作目录，你可以在 NSSM 的界面中配置这些选项。

当你的应用程序作为服务运行时，你可以随时在 Windows 服务管理器中对其进行启动、停止、重启等操作
